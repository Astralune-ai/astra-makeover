# Supported design styles

Pass as `--brief Style=<name>`. Default is **Modern Australian**.

**Core (good for resale across most Sydney/AU markets):**
Modern Australian · Contemporary · Scandinavian · Japandi · Minimalist · Organic Modern ·
Coastal · Hamptons · Modern Farmhouse · Warm Minimal · Classic Contemporary

**Also supported:**
Industrial · Mid-century Modern · Luxury Contemporary · Wabi Sabi · Dark Contemporary ·
California Modern · Nordic

**Named references (use as the style cue):**
Hotel · Resort · Apple Store · Muji · Four Seasons · Aman · Soho House — or any custom style
the client names. A Pinterest / Houzz / Instagram reference image can be passed as an extra
`--ref` and overrides the written style.

Pick by goal: **Modern Australian / Coastal / Hamptons** read safest for broad resale appeal;
**Japandi / Warm Minimal / Organic Modern** read as higher-design for premium buyers.
